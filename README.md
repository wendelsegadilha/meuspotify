# Meu Spotify
Acesse aqui: [meuspotify](http://wendelsegadilha.xyz/meuspotify/)

### Funcionalidades
- Cadastro de usuários
- Criação de Playlists
- Gerenciamento de Playlists
- Upload de arquivos MP3
- Reprodução das Playlists

### Tecnologias utilizadas: 
- JSP
- Servlets
- MySQL
- HTML
- CSS
- JavaScript
- jQuery
- Bootstrap
